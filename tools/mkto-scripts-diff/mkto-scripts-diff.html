Marketo form scripts diff

Compares extracted scripts between two forms (getForm JSONP). Host, munchkin ID, and page URL match the scripts inspector. Use ?formA= and ?formB=. Identical scripts are collapsed by default. Changed diffs open at the top of the file; use Prev change / Next change (or focus the bar and use ← → or P / N) to move between hunks—Prev from the first hunk returns to the top. Results: window.mktoScriptsDiff.
